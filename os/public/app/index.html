Moje zlecenia (dziś)

    Szczegóły zlecenia

    Wpisz ID lub wybierz z listy / tabeli, potem „Pobierz”. Status — wartości zgodne z API (Nowe, Zaplanowane, W_Realizacji, Zakonczone).

    Edycja zlecenia

    Przy dodatkowej pracy zaktualizuj wartość rzeczywistą i opis zakresu. Wartość planowana może się zmienić (np. po uzgodnieniu z klientem).

    Rozpoczęcie pracy (ekipa)

    Wymagane: GPS + checklista + BHP. Użyj „Pobierz GPS” przed wysłaniem.

    Zdjęcie z GPS

    Katalog pojazdów (import)
