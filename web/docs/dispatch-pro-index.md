# Dispatch PRO - Indeks dokumentow

Jeden punkt startowy dla zespolu (operacje + UAT + materialy do wydruku).

## Dokumenty glowne

- `web/docs/dispatch-pro-uat-runbook.md`
  - pelny scenariusz UAT i runbook operacyjny
  - testy end-to-end, diagnostyka i szybkie debugowanie

- `web/docs/dispatch-pro-go-live-onepager.md`
  - skrocona checklista go-live (wersja szybka)

## Materialy do wydruku

- `web/docs/dispatch-pro-go-live-a4.md`
  - checklista A4 po rolach (PL)

- `web/docs/dispatch-pro-go-live-a4-bilingual.md`
  - checklista A4 dwujezyczna (PL + RU/UA)

- `web/docs/dispatch-pro-mini-card.md`
  - mini karta 5 krokow na stanowisko

## Kolejnosc wdrozenia dla zespolu

1. Przeczytaj `dispatch-pro-go-live-onepager.md`
2. Wydrukuj:
   - `dispatch-pro-go-live-a4.md` lub `dispatch-pro-go-live-a4-bilingual.md`
   - `dispatch-pro-mini-card.md`
3. Wykonaj testy z `dispatch-pro-uat-runbook.md`

## Kto z czego korzysta

- Wyceniajacy:
  - onepager + mini karta
- Specjalista:
  - onepager + A4 + UAT runbook
- Brygadzista:
  - A4 + mini karta
- Koordynator wdrozenia:
  - UAT runbook (pelny)
